# Deployed Dashboard
The dashboard has been deployed on Streamlite and can be accessed via this link:
https://averageuserid-career-outcome-dashboard-app-brnlpi.streamlit.app/

# Summary
The goals of the project are to:
1. Evaluate and integrate public data sources for career outcome data at the national level so related inqueries could be answered in a consistent and reliable way
2. Collect and visualize career outcome data for B.A. in Spanish Language and Literature, M.A. in Foreign Language Teacher Education, and Ph.D. in Digital Humanities

# What is in This Repo?
1. Codes used to deploy the dashboard
2. An asset folder containing a table evaluating various public data sources from various angles
3. A data folder containing unprocesses data taken from IPEDS
